
# Week 6 Employee Frontend README

## Employee Management Frontend using React.js, Tailwind CSS, React Router, and Axios

---

# Overview

This project focuses on frontend development for an Employee Management System using React.js and Tailwind CSS. The frontend communicates with backend REST APIs and provides employee management functionalities through a responsive user interface.

The project demonstrates component based architecture, routing, API integration, form handling, authentication flow, and state management.

---

2. Vite

3. JavaScript ES6

4. Tailwind CSS

5. React Router DOM

6. Axios
