## HTML Webpage Structure, Lists, Images, Links, and Basic Styling

---

# Overview

This assignment focuses on learning the fundamentals of HTML and webpage creation. The project demonstrates how to structure webpages using HTML tags, add headings, paragraphs, images, links, lists, and apply basic inline styling.

The webpages simulate simple static websites using core HTML elements and formatting techniques.

This week also introduced webpage structuring concepts and element nesting for better content organization.
