# Overview

This assignment focuses on frontend web development using HTML, CSS, Tailwind CSS, and React.js. The project demonstrates responsive webpage layouts, utility first styling with Tailwind CSS, table creation, navigation design, reusable React components, props handling, JSX rendering, and component based UI development.

Tailwind CSS was newly introduced in this week for fast and responsive UI styling using utility classes.

React.js was also introduced for building reusable and dynamic frontend components using JSX and props.

# How to Use

## Install Dependencies

1. Open terminal inside project folder

2. Run:

```bash
npm install
```

---

## Run Tailwind CSS Project

1. Start Tailwind CSS build process

```bash
npm run dev
```

2. Open HTML file in browser

---

## Run React Application

1. Open React project folder

2. Run:

```bash
npm install
```

3. Start development server

```bash
npm run dev
```

4. Open browser using generated localhost URL

# New Concepts Introduced

1. Tailwind CSS

   Utility first CSS framework used for rapid UI development using predefined classes.

2. React.js

   JavaScript library used for building reusable component based user interfaces.

3. JSX

   Syntax extension that allows writing HTML like code inside JavaScript.

4. Props

   Mechanism used to pass data between React components.

5. Vite

   Fast frontend build tool introduced for React project setup and development.

6. Flexbox

   CSS layout system for arranging items in rows and columns.

7. CSS Grid

   Two dimensional layout system used for responsive webpage structures.
