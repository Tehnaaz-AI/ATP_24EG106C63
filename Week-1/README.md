# Overview

This assignment demonstrates the fundamentals of JavaScript programming using arrays, loops, functions, conditional statements, and object manipulation. The programs focus on solving basic logical problems and performing different operations on arrays and employee objects.

The assignment helps in improving problem solving skills and understanding core JavaScript concepts.

---

# Technologies Used

1. JavaScript
2. Node.js
3. Browser Console

---

# Programs Included

1. Find the Smallest Element in an Array

2. Sum of Array Elements

3. Sum of Array Elements Using Function

4. Largest of Two Numbers

5. Largest of Three Numbers

6. Largest of Three Numbers Using Function

7. Search an Element in an Array

8. Employee Array Operations Using `splice()`

---

# How to Use

## Using Browser Console

1. Open any browser
2. Press `F12` to open Developer Tools
3. Open the Console tab
4. Copy and paste the JavaScript code
5. Run the program

## Using Node.js

1. Install Node.js
2. Save the file as `week1.js`
3. Open terminal or command prompt
4. Run the following command:

```bash id="q4zh3j"
node week1.js
```
