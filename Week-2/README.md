# Overview

This assignment focuses on advanced JavaScript concepts including ES6 features, array methods, object manipulation, classes, modules, asynchronous programming, and backend development using Express.js.

The programs simulate real world applications such as e commerce systems, task management systems, library management, banking systems, and REST APIs. The assignment improves modular programming, backend development, and problem solving skills.

# How to Use

## Using Browser Console

1. Open any browser
2. Press `F12` to open Developer Tools
3. Open the Console tab
4. Copy and paste the JavaScript code
5. Run the program

---

## Using Node.js

1. Install Node.js

2. Install dependencies:

```bash id="tpr20x"
npm install
```

3. Start the server:

```bash id="j67v75"
node server.js
```

4. Open browser or API testing tool and use:

```text
http://localhost:3000
```
